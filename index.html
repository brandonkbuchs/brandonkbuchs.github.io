Portfolio

Park Mobile Application

Spring 2020

Duration: 60 Hours

I built a mobile applicaiton using react-native and Google's firebase. The application is designed to assist visitors to Bryce Canyon National Park enjoy their experience by accessing points of interest, writing reviews, and reporting hazards. Additionally, users can search reviews and customize the visible features of the map. The application uses volunteered geographic information in user location, user authentication, and cached database queries from Cloud Firestore.

Cancer Correlation Analysis

Spring 2020

Duration: 40 Hours

This web map allows users to explore interpolated data to view a possible correlation between nitrate concentrations and discrete sample locations and cancer rates recorded by census tract in the state of Wisconsin. The project incorporated Leaflet.js, HTML, CSS, and Turf.js, an open source spatial analysis library for Javascript

Georgia's Trail Problem

Fall 2018

Duration: 40 Hours

I made this infographic as an assignment in a graphic design in cartography course. The assignment's goal was to become familiar with bivariate data analysis and Adobe Illustrator. I examined the relationship between trail density and population growth in the state of Georgia to produce an infographic that advocates for more trail construction in Georgia. I used ArcGIS, Python, and Adobe Illustrator to create this infographic.
